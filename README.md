# Real-Time Multi-Coin Crypto Dashboard (MVP)

## Features
- Streams BTC, ETH, and DOGE prices live from Binance
- Multi-coin selector
- Live updating chart with Recharts
- Node.js backend and React frontend

## Quick Start

### Backend
\`\`\`bash
cd backend
npm install
node server.js
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`

Open http://localhost:9000
